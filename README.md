## Description
A web application built using Python and Streamlit that allows users to upload resumes and receive a basic resume analysis score.

## Features
- Enter user name
- Upload PDF resume
- Display resume details
- Resume scoring
- Interactive web interface

## Technologies Used
- Python
- Streamlit
